# Changing Favicon 

Go to [Favicon Generator](https://www.ionos.ca/tools/favicon-generator#tool) and upload your big favicon logo. The site will generate different version of icons.

Upload all those images on blogger and place them in following code. 

import { Callout } from 'nextra/components'
 
<Callout type="info" emoji="ℹ️">
  You can create a blogger post and upload all your images there. You can get image address from there. Do not publish that post.
</Callout>

<Callout type="warning" emoji="⚠️">
  Please place proper images to proper sizes.
</Callout>
